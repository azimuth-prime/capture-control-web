The details below display the current state of this audit. For audits that have been completed, it will display a list of all the items contained within the audit that had stock discrepancies only. Items that were accurate will not be displayed in the list.

For currently open audits, the list will only contain the audit inventory items that have not yet been counted and updated in the system. The list can be viewed in two modes 1) Mobile Mode where the items are listed in a page by page basis. In order to move to the next page you will need to complete the count on the current page. 2) Print mode where all results are available to print in pages that can be passed to staff for counting.
