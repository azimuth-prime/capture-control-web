Sales Tax

Add, edit and delete system wide sales taxes.

Sales tax

If using the Intuit integration, match the BusinessHub tax that you have defined with a matching tax from the Intuit system. The exempt sales tax is a system default and cannot be edited or deleted. If using Intuit, then match the Exempt tax with the Intuit Exempt tax.

The ISO Code value is structured using the 2 character ISO codes for country and province/state. ie: Ontario, Canada would be "CA-ON". The "Tax Name" value is the name that is displayed on all invoices and communications in the summary sections. ie: "GST", "PST", "VAT", etc. Order of Operations controls how taxes are applied in cases where they are stacked. By default this is left blank.
