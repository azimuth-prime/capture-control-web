Data Import

Import data into the system using the provided excel templates. The template downloads provide a description of the fields within the excel template as well as the excel template itself. Items that are "required" are marked with a *. After upload the data will be presented for editing and approval prior to final import.

Import Products

Import via Excel file, product information. If you use a single SKU model then make sure that you have that set in the product configuration section. When products are imported a generic SKU will be created for each.

Select the file for import. You can download the template by clicking on the link to the right.

Import SKUs

Import via Excel file, SKU information.

Select the file for import. You can download the template by clicking on the link to the right.

Import Inventory Lots

Import via Excel file, inventory/lot information.

Import File

Verify and adjust the data below. Once it has been checked you can complete the upload process by clicking the 'Confirm Upload' button at the bottom.

Field Set:

  • id (Optional - default = System Generated Id)
  • Name ( * )
  • altId (Optional)
  • description (Optional)
  • availabilityDate (Optional - Default value = Now)
  • taxable (Optional = default value = Yes)
  • controlType (Optional - default value = LOT)
  • stockUnit (Optional - default value = EACH, Make sure that stock units used are defined in the system)
