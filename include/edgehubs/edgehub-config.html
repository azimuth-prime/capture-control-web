RFID Tag Printing
Setup the printing configuration for RFID Tags. RFID Printing requires an Edgehub device assigned to a physical warehouse. If an inventory lot belongs to a physical warehouse that does not have an Edgehub device then RFID tag printing will be disabled.
Enable the use of a custom RFID Tag. If 'Yes' then the ZPL II code is required below.
The Edgehub integrates with the Zebra line of RFID printers and uses the ZPL II code language for printing. Enter your ZPL II code in the below text area to use your own template for printing. If left blank then the default Edgehub template will be used. The default Edgehub template can be viewed within the Edgehub Admin Interface.
